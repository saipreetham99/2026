# --- NOT on PyPI: your local hardware driver files ---

# PYTHONPATH or copied next to rov-server.py):
#
#   pca9685    -> your ~/drivers/pca9685-python/pca9685  (thruster PWM)
#   icm20602   -> your icm20602.py driver               (gyro yaw)
#   ms5837     -> BlueRobotics ms5837-python (git clone) (depth)
#                 https://github.com/bluerobotics/ms5837-python
#
# Nothing to install for these — just confirm `python -c "import pca9685, icm20602, ms5837"`
# succeeds on the Pi before running the server.
